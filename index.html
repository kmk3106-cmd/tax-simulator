용어 정리
• 결정세액 = 1년간 본인이 부담하는 세금 총액 (산출세액 + 지방세, 세액공제 미반영 가정)
• 종합과세로 인한 추가세액 = 배당이 종합과세에 들어와서 늘어난 절대금액 (분리과세였다면 안 냈을 부분)
• 배당 총 부담률 = 배당 1원당 떼이는 세금 비율 (배당 총 부담세금 ÷ 배당)
• 배당 추가 부담률 = 분리과세 15.4% 대비 추가로 더 부담하는 % (= 총 부담률 − 15.4%)
• 근로소득 원천징수 = 회사가 매월 떼고 연말정산으로 정산한 세금 (≒ 근로 단독 결정세액)
• 배당소득 원천징수 = 배당 받을 때 15.4%로 자동 차감
• 5월 실제 납부액 = 결정세액 − 기납부세액 = 본인이 통장에서 추가로 국세청에 보내는 돈 (음수면 환급)
    수학적으로 "5월 실제 납부액 = 종합과세로 인한 추가세액" 이 정확히 일치합니다.

※ 본인 인적공제(150만) 외 다른 소득공제 없음 가정. 근로소득세액공제·배당세액공제(국내배당)·외국납부세액공제(해외주식) 등은 미반영. 배당이 2,000만원 이하면 분리과세 종결되어 종합신고 의무 자체가 없으므로 5월 추가 납부 0원.

2026년 직장가입자 부과 규칙
• 건강보험료율 7.19%, 장기요양보험료 = 건강보험료의 12.95%
• 보수월액보험료 = 월급 × 7.19% — 회사 50% / 본인 50% 매월 자동 납부
• 소득월액보험료 = (보수외소득 − 2,000만) × 7.19% — 본인 100% 부담 (보수외소득 연 2,000만 초과시)
• 단, 이자·배당소득이 1,000만원 이하면 보수외소득에서 제외 → 소득월액보험료 영향 없음
• 모든 보험료에 장기요양보험료(건보료의 12.95%)가 추가 부과됨
